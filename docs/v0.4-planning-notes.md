## The headline tension (resolve first in brainstorm)

User leans AI-BOM. But the research makes AI-BOM look thinner than expected (poor SPDX fit), while the **policy engine** is meatier/better-specified and **deep-scan** gives real detection depth (TruffleHog parity). So v0.4's headline is a genuine three-way:

- **AI-BOM** → compliance artifact for enterprise buyers (CISA Dec-2025 guidance endorses MITRE ATLAS, which we already map)
- **Policy engine** → governance / anti-compliance-theater (enterprise adoption)
- **Deep-scan** → detection depth / TruffleHog parity (security-engineer appeal)

Supporting scope (the smaller v0.3-deferred verification items) likely ships regardless of which headline wins.
